Solving To Lower Case

Given a string s, return the string after replacing every uppercase letter with the same lowercase letter.


Example 1:

Input: s = "Hello" Output: "hello"

Example 2:

Input: s = "here" Output: "here" Explanation: Product of digits = 4 * 4 * 2 * 1 = 32, Sum of digits = 4 + 4 + 2 + 1 = 11, Result = 32 - 11 = 21.

Constraints:

1 <= s.length <=100 s consists of printable ASCII characters.
